Getting Started — Firebase Setup

1. Create a Firebase projectGo to console.firebase.google.com → Add project
2. Enable FirestoreBuild → Firestore Database → Create database → choose Test mode
3. Get your configProject Settings → General → Your apps → Web app → copy apiKey and projectId (or copy the entire configuration block!)
4. Paste & ConnectOpen Settings, paste keys, and save. **Firestore Rules Note**: In Test mode, rules expire in 30 days. Make sure your Rules look like `allow read, write: if true;` or similar to allow access without authentication.
